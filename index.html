Menyimpan gambar di HTML

  • Untuk menyimpan gambar di HTML, kita bisa menggunakan tag image dengan atribut src didalamnya. contoh: < img src="" >
  • Terdapat 2 cara untuk menampilkan gambar di file HTML kita tergantung dari sumber filenya

  1. Internal resource
    • Jika file gambar yang ingin digunakan berada didalam folder yang sama dengan file HTML yang digunakan, Maka value di atribut src nya cukup tuliskan nama filenya dan ekstensi filenya. INGAT ekstensi file wajib ditulis
    • Misal saya ingin menampilkan file gambar yang bernama bakso.jpg . Maka full tagnya menjadi: < img src="bakso.jpg" >

  2. Eksternal resource
    • Menampilkan gambar dari website lain tidak disarankan untuk menggunakan external karena:
      1. jika menggunakan gambar dari website lain, maka kita menggunakan bandwith website lain ketika user mengakses gambar tersebut di web kita. curang dong ya.
      2. kita harus memastikan apakah gambar tersebut bebas copyright atau tidak
      3. bergantung ke website lain. jika web lain down/tidak bisa di akses. maka gambar di web kita pun akan error

Atribut di tag img

  1. src: merupakan atribut wajib untuk menentukan lokasi file gambar

  2. alt : berguna untuk memberikan alternatif text pada gambar. alt ini akan muncul ketika gambar yang kita tampilkan error/tidak bisa dibuka di device user
    cohtoh < img src ="bakso.jpg" alt="gambarbakso" >
    Jika gambar error/tidak tidak bisa muncul di device user, maka yang akan alternatif teks nya, seperti contoh dibawah ini


  3. title : untuk memberijudul gambar. jadi ketika user mengarahkan kursor ke gambar yang ditampilkan, maka akan muncul tulisan title nya
    contoh < img src="bakso.jpg" title="Bakso Mang Ujang" >
    Silahkan arahkan kursor ke gambar dibawah ini

  4. width dan height : untuk mengatur lebar dan tinggi gambar
    untuk value dari artibut width dan height, bisa menggunakan 2 nilai. jika hanya menuliskan angka, maka satuannya berbentuk pixel. akan tetapi jika ditambahkan % maka ukurannya relatif terhadap halamannya.
    misal jika mengatur width di 50% maka ukuran lebar gambar akan menyesuaikan dengan ukuran halaman (setengah dari halaman)
    jika halaman browser diper besar atau diperkecil, maka ukuran gambar akan menyesuaikan.
    contoh: < img src="bakso.jpg" width="50%" >
